# AI Study Assistant 📚🤖

An AI-powered study assistant built with Flutter, Firebase, and OpenAI.

## Features (planned)
- 💬 AI Chat
- 📄 PDF summarization
- 🧠 Quiz generation
- 🔐 Authentication
- 📊 User history

## Tech Stack
- **Frontend**: Flutter
- **Backend**: Firebase (Auth, Firestore, Storage)
- **AI**: OpenAI API

🚧 In active development
